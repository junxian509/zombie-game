# Zombie Game

双人同屏打僵尸网页小游戏。两名玩家在同一台电脑上移动、射击、清理波次，并可在合作模式和对抗模式之间切换。

## 在线文件

主游戏文件是 `index.html`。这是一个单文件 Canvas 游戏，HTML、CSS、JavaScript 和兜底绘制资源都内嵌在文件中，不需要安装依赖或启动服务器。

## 如何运行

1. 下载或克隆仓库。
2. 用浏览器直接打开 `index.html`。
3. 进入游戏后按键即可开始操作。

也可以在本地项目目录中打开：

```bash
open index.html
```

## 游戏模式

### 合作模式

- 默认模式。
- 两名玩家一起抵御僵尸。
- 清空当前波次后自动进入下一波。
- 任意玩家死亡则游戏失败。
- 清完第 10 波后人类获胜。

### 对抗模式

- 点击右上角模式按钮切换。
- 两名玩家各自计算击杀数。
- 每波结束时，击杀数较低的一方扣 1 点生命，最低保留 1 血。
- 清完一波后需要点击“下一波”继续。
- 一方生命归零时，另一方获胜。

## 操作方式

| 玩家 | 移动 | 射击 |
| --- | --- | --- |
| 玩家1 蓝色 | `W` `A` `S` `D` | `F` |
| 玩家2 红色 | 方向键 | `.` 或 `/` |

其他按键：

- `R`：重置游戏
- 右上角按钮：切换合作/对抗模式
- 对抗模式中清波后：点击“下一波”

## 核心机制

- 第 1 波生成 7 只普通僵尸。
- 每波总数 = `5 + 当前波次 * 2`。
- 第 3 波开始出现精英僵尸。
- 精英僵尸替代普通僵尸，不额外增加总数。
- 精英僵尸需要 5 枪击杀，移动更慢，碰撞伤害更高。
- 同屏僵尸数量上限为 20 只。
- 玩家射击有冷却限制，每秒最多约 3 发。

## 自动测试

`index.html` 内置了测试入口。打开下面地址即可运行自检：

```text
index.html?autotest=1
```

测试覆盖内容包括：

- 双人独立移动和射击
- 波次生成规则
- 精英僵尸生命和血条
- 普通/精英碰撞伤害
- 模式切换和波次推进
- 游戏重置
- 僵尸数量上限和性能裕量
